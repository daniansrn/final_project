# Scraps to Snacks – Recipe Generator

## Project Overview

For this project, I made a recipe generator that users can use to turn their leftovers into food so they don’t waste them. The way it works is that the user will enter their leftovers (for example: milk, banana, or any other food that is about to expire), and the AI will generate suitable recipes based on those ingredients using the Edamam Recipe API.

---

## Features / Scenarios

### Enter leftovers
The user will enter any leftovers they have into the chatbox.

---

### Search recipes
The AI will fetch suitable recipes based on the ingredients entered using the API and give recipe recommendations to the user.

---

### View nutrition info
The user will be able to see nutrition information for each recipe based on the data provided by the API.

---

### Ingredient substitution
If the user is missing any ingredients, the substitution agent will read from a substitution database and suggest alternatives.

---

### Allergen info
The system will provide allergen information to help users identify any potential allergens in the recipe.

---

### Rate recipe
This feature allows users to rate recipes. Higher-rated recipes will be prioritized and shown first in future search results.
